FREQUENTLY ASKED QUESTIONS

AVONIC is a localized vermicomposting system that automates environmental control using sensors and an ESP32-powered dashboard for small-scale farms.

It monitors soil moisture, gas, and temperature, and automatically adjusts humidity while displaying live data on a web dashboard.
